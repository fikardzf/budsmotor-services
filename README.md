# BUDS MOTOR Website

Static single-page website untuk katalog motor multi-brand:
Honda, Yamaha, Suzuki, Kawasaki, dan Vespa.

## Running Local

```bash
npm install
npm run dev
```

Buka `http://localhost:5173/`.

## Build Production

```bash
npm run build
```

Hasil production ada di folder `dist/`.

## Netlify Settings

- Build command: `npm run build`
- Publish directory: `dist`
